Computer Programming:
In this class you will learn the fundamentals of computer programing in various languages starting with C++. You will learn the basics of functions, classes, methods, objects, variable types, and how different languages operate while retaining the same fundamental possesses, with syntax being the main difference.

Professor: Stephen Graham

Computer Networking I:
In this class you will learn the fundamental skills related to networking. You will learn how the internet works, the TCP/IP protocol suite that powers the internet as we know it. You will learn the differences between IPv4 and IPv6 and there use case, and the how Switches, Routers, WAPs, and other networking devices work.

Computer Mathamatics:
In this class you will learn the basic mathematic processes that power computing. You will learn binary, hex, and octal numbering systems and math. You will learn about matrix math, and how to do basic mathematics and solve algebra with matrixes. You will learn the fundamental mathematic building blocks that enable all other classes to do the things they do with computers
